# The Fikri System

The platform follows a Fikri (Intellectual) approach to learning.

Knowledge is not treated as isolated facts.

Instead, every topic progresses through layers:

### 1. Exposure

Understanding what exists.

### 2. Comprehension

Understanding how it works.

### 3. Integration

Connecting it with previously learned knowledge.

### 4. Application

Using it in clinical and examination settings.

### 5. Retention

Ensuring it remains available months and years later.

### 6. Mastery

Reaching the point where recall becomes automatic.

The goal is not completing chapters.

The goal is transforming information into clinical intuition.

# Science-Based Learning Architecture

MedTrack Pro combines proven learning principles:

### Spaced Repetition

Topics appear exactly when the brain is most likely to forget them.

### Active Recall

Knowledge is retrieved from memory rather than passively reread.

### Interleaving

Subjects and concepts are mixed strategically to improve retention.

### Deliberate Practice

Weak areas receive more attention than comfortable ones.

### Reflection

Every study session creates feedback for future improvement.

# The Chapter System

The human brain thinks in concepts, not file structures.

Therefore:

* Subjects contain chapters.
* Chapters contain topics.
* Topics drive the revision engine.

Each chapter acts as a command center.

Within every chapter you can:

* Track all topics
* Mark high-yield concepts
* Identify weak areas
* Monitor progress
* Review learning resources
* Maintain clinical notes

The chapter provides clarity.

The topics provide precision.

# Revision Is Sacred

Most forgetting happens because revision is random.

MedTrack Pro treats revision as a scientific process.

Every review session answers one question:

The system continuously adjusts future review dates based on performance.

Time is invested where it creates the greatest improvement.
